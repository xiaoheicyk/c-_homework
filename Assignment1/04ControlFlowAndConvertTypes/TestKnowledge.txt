4. What is the difference between x = y++; and x = ++y;?
x = y++: x not change, y increases by one;
x = ++y: x and y both increases by one.

5. What is the difference between break, continue, and return when used inside a loop
statement?
The break statement is used to exit the loop immediately.
The continue statement skips the current iteration of the loop and jumps to the next iteration.
When a return statement is executed, the method terminates immediately, and any remaining iterations 
in the loop are not executed.
